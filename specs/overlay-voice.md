# Clarity — Overlay Voice & UX

## Visual rules

- Highlight color: purple glow only (#7c3aed)
- Comment bubble: white card, purple left border, inline next to element
- Never use red or yellow highlights — those feel like errors, not insights
- Dismiss button must be per-card, never a "dismiss all"

## Comment structure

- Label: "👁 Clarity" — always exactly this, never changed
- Flag: witty observation, max 15 words
- Suggestion: italic, purple, one action
- Action button: only shown when a concrete action exists (uncheck, open_tab)

## Prohibited UI patterns

- No modal overlays that block content
- No sounds or vibration
- No animations longer than 300ms
- No persistent UI that cannot be dismissed
