Real-time detection
Our Chrome extension scans the DOM continuously for behavioral manipulation patterns — countdown timers, scarcity messages, social proof, infinite scroll, and more.
02
AI-powered commentary
Claude generates witty, plain-language explanations tailored to the specific site and pattern — not technical jargon, but the kind of thing a sharp friend would whisper to you.

By naming manipulation out loud, Clarity creates a moment of friction — a beat of self-awareness before you click, buy, or keep scrolling.

How It Works

Three steps. Zero friction.

1
Detect
Content script scans DOM for known manipulation patterns using CSS selectors and text regex — continuously, as the page updates.
→
2
Translate
Matched pattern + site context is sent to Claude API. Claude generates a sharp, human, often humorous 1–2 sentence observation.
→
3
Surface
A Clarity card appears in the bottom-right corner. The user sees it, reads it, and — crucially — pauses. Then it disappears on its own.

Technical Architecture

Built to ship
in hours.

Frontend Layer
⚙
Chrome Extension MV3
Content script for DOM scanning. MutationObserver for dynamic content. Service worker for API calls.
🎨
Injected Overlay UI
Clarity cards animate in/out with zero dependency on page styles. CSS scoped under #clarity-container.
🔍
Pattern Detector
CSS selector matching + regex text scanning across 5 core manipulation categories.
Intelligence Layer
🤖
Claude claude-sonnet-4-20250514
System-prompted to generate sharp, site-specific, 1–2 sentence observations with personality and wit.
⚡
Context-Aware Prompting
Pattern type, matched text, and site hostname all fed to Claude for precise, non-generic output.
🛡
Graceful Fallback
Pre-written fallback comments ensure the overlay still works if the API is unavailable.
